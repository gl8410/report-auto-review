# 1.1.2 建设项目必要性、选址合理性及前期工作内容

# 1.1.2.1 建设项目必要性

312国道作为国省干线骨架路网，是沪宁发展轴线上的重要交通基础设施，也是无锡市国省干线骨架路网的重要组成部分，312国道改造以及其他国省干线等交通基础设施的建设是对新一轮城市总体规划的重要响应，将对城市发展建设起到重要的支撑和带动作用。

本项目段为凤翔路至广石路段，路线起自现状312国道与惠山区凤翔路交叉处西侧，沿老路向西，止于与惠山区广石路交叉处，路线全长约  $4\mathrm{km}$  。

本项目针对受312国道高架桥桩基、匝道桥桩基、引道挡墙、地面桥结构影响（与结构物碰撞或距离结构基础较近）或覆土不满足要求的电力排管进行迁改，对受312国道改造影响的  $110\mathrm{kV}$  及以下的电力架空线进行入地改造， $220\mathrm{kV}$  电力架空线迁改采用架空改架空形式，本次工程为  $10\mathrm{kV}$  电力管线因此进行入地改造。

# 1.1.2.2 选址合理性

本次设计原则：对改造后位于312国道道路、桥梁范围内的架空杆线进行迁移或入地改造，将惠暨大道～振石路段道路北侧受新建桥墩影响的电力排管进行改移新建。

312 国道跨越锡澄运河处拆除现状惠澄大桥，扩建主线高架+两侧辅道桥梁型式跨越锡澄运河，为简支钢桁架结构，主线高架与辅道桥共用桥墩。跨河桥梁跨径为  $2 \times 28.6 + 125.0 + 52.5\mathrm{m}$  （自东向西），桥梁走向与河道水流方向夹角约为  $90^{\circ}$ ，东西两岸各采用4个桥墩顺水流向布置，跨河桥梁横断面宽度为  $79.25\mathrm{m}$  （沿水流方向）。

# 1.1.3 建设项目涉河工程建设方案

# 1.1.3.1 建设方案总体布局、规模

本工程下穿锡澄运河，下穿入土点DL49位于河道西岸，工作井与现状河口线最小距离为  $77.8\mathrm{m}$ ，与规划新建护岸边线最小距离为  $71.6\mathrm{m}$ ，与背水侧堤脚线最小距离为  $60.5\mathrm{m}$ ，坐标为X：3501671.011，Y：525310.020；下穿出土点DL47位于河道东岸，工作井与现状河口线最小距离为  $145.3\mathrm{m}$ ，与规划新建护岸边线最小距离为  $150.5\mathrm{m}$ ，与防汛道路边线最小距离为  $133\mathrm{m}$ ，坐标为X：3501586.320，Y：525695.836。

穿越段采用定向钻穿越锡澄运河，出、入土点之间水平长度  $395.0\mathrm{m}$ ，与河道水流方向交角为  $90^{\circ}$ ，管道入土角为  $18^{\circ}$ ，出土角为  $19^{\circ}$  。电力管线管顶标高  $-11.583\mathrm{m}$ ，距离现状河底  $9.509\mathrm{m}$ ，距离规划河底  $11.083\mathrm{m}$ 。

# 3、定向钻施工方案

# （1）准备工作

1）施工前先对现场进行地质勘探，地质勘探主要了解有关地层和地下水的情况，为选择钻进方法和配制钻液提供依据。其内容包括：土层的标准分类、孔隙度、含水性、透水性以及地下水位、基岩深度和含卵硕石情况等。可采用查资料、开挖和钻探方法获取。  
2）结合业主提供的资料，弄清施工区域范围内有无地下管线，并弄清地下管线的种类、数量、走向和高程，为设计钻进轨迹提供依据，以防施工受损。勘察设计管道所穿越的位置是否有足够的空间，以满足钻机固定位置和导向所需。  
3）本工程所用材料、管道附件的材质、规格、型号必须符合设计要求。同时应满足供货合同中的技术条件和相关标准的要求。材料、管件应具有出厂合格证，主要材料和管件还应有监制人员确定的质量证明书，上述文件应满足可追溯性要求。

# (2) 钻孔曲线、测量放线

根据施工图纸，进行测量放样。并根据施工范围的地质情况、埋深、管径确定管材和一次定向钻进的管道长度，并设计好钻杆轨迹。

高程控制根据勘测方提供的水准点引测施工现场的高程控制点。根据本工程的实际情况，在现场选择固定的地方做临时水准点，并做好保护。

# (3) 出入土坑开挖

土方施工队可以根据已布设好的井坑位置开槽施工其入钻点井坑，以及出钻点井坑。

# （4）钻机就位

钻机运到现场必须先锚固稳定，钻机如果锚固不稳，将会发生功率损失或者功率作用在机器身上，造成机器和人的伤害。钻机是依靠地锚坐和后支撑与地基固定的，安放钻机前应先平整场地，根据预先设计的钻机倾斜角度进行调整，依靠钻机动力将锚杆打入土中，使后支撑和前底座锚与地层固结稳定。

钻机就位后，调整钻机导向杆到略高于设计管位中心高程的位置，水平钻入土中。在导向钻头中安装发射器，通过地面接收器，测得钻头的深度、鸭嘴板的面相角、钻孔顶角、钻头温度和电池状况等参数，将测得参数与钻孔轨迹进行对比，以便及时纠正。地面接收器具有显示与发射器功能，将接收到的信息无线传送至钻机的接收器并显示，操作手根据信号反馈纵向钻机按正确的轨迹钻进。

在导向钻机过程中技术人员根据探测器所发回的信号，判断导向位置与钻进路线图的偏差，随时调整。并把调整数据值记录在“钻进位置”相应的表格中。

为了保证导向头能严格按照操作人员发出的指令前进，需要在管道线路初步布点对控制点进行加密加细。间隔  $10\mathrm{m}$  设中线、高程控制点，用木桩做出明显标志，并在桩点周围用混凝土砌出护墩加以保护。控制人员严格按照点位操纵仪器。

# （5）钻液的配置

钻液的好与坏对于拉管施工的成败起到了极关键的作用。钻液具有冷却钻头、润滑钻具，更重要的是可以悬浮和携带钻屑，使混合后的钻屑成为流动的泥浆顺利地排出孔外，既为回拖管线提供足够的环形空间，又可减少回拖管线的重量和阻力。残留在孔中的泥浆可以起到护壁的作用。泥浆粘度现场测量采用马氏漏斗，每2小时测量一次。

定向钻穿越施工要求泥浆的性能高，泥浆的性能主要有失水以及润滑的性能。根据施工检验，我们采取以下措施：

1）按照事先确定好的泥浆配比用一级膨润土加上泥浆添加剂，配出合乎要求的泥浆。  
2）使用的泥浆添加剂有：将失水剂、提粘剂和防塌剂等。所添加剂符合环保。  
3）为了确保泥浆的性能，使膨润土有足够的水化时间，在用量不改变的情况下，采取两套泥浆贮存罐，延长循环周期的措施。

据地质土层的不同，泥浆的配比也随之变化，并选用不同的添加剂，以达到预期的效果。基浆的配置：5—8%预水化膨胀土，碱搅拌后水化而成（充分水化能明显提高泥浆的性能），碱的用量根据水质情况具体确定。在各个阶段的配置方法如下（加量按重量比计算），这些泥浆配置方案都是针对粘质粉土提出的；如果地质情况有变化，其配置方案也随之变化。

1）导向孔阶段要求尽可能将孔内的泥沙携带出孔外，同时维持孔壁的稳定，减少推进阻力；其基本配方是：基浆  $+0.2 - 0.4\%$  增粘剂  $+0.3\%$  将滤失剂。  
2）扩孔阶段要求泥浆具有很好的护壁效果，防止地层塌方，提高泥浆携带能力；其基本配方为：基浆  $+0.3 - 0.4\%$  提粘剂  $+0.4\%$  将滤失剂。

3）扩孔回拖阶段要求泥浆具有很好的护壁、携砂能力；同时还有很好的润滑能力，减少摩阻和扭矩；其基本配方如下：基浆  $+0.3 - 0.5\%$  提粘剂  $+0.4\%$  将滤失剂  $+2 - 3\%$  的润滑剂。  
4）为了有效地减少钻杆与地层之间的摩擦阻力，泥浆的泥饼的质量很重要，泥饼薄而坚韧能够稳定孔壁，减小摩阻。我们在泥浆中加入高性能的将滤失剂，控制水分，形成高质量的泥饼。

# (6) 测量和沉降观测

定向钻施工的穿越轨迹上方要布置地面沉降观测点。出洞后的  $15\mathrm{m}$  范围内每隔  $3\mathrm{m}$  布置一个沉降观测点，然后每隔  $5\mathrm{m}$  布置一个沉降观测点，在地面重要构筑物处适当增加沉降观测点布置，在定向钻穿越轨迹上方还要布置几道断面沉降观测点。沉降观测点布置必须做到安全、可靠，对测量所有的仪器要经常校正，确保测量数据的准确性。

定向钻施工时，对沉降观测点每天进行4次观测，对要保护的构筑物处的观测点适当增加观测的次数。并及时把测量数量反馈到施工人员手中，以便根据沉降量，调整施工参数，便于指导牵引施工。从导向开始，如任一观测点累计沉降超过1cm，停止施工，查找原因，提出修改方案，报高速相关部门审批后再进行施工。扩孔结束，拉管时根据管道头行进位置，进行跟踪测量，尤其对管头进入高速边坡和出高速边坡时，如有观测点沉降超0.5cm时，停止施工，处理方案同上。钻至道路附近时，应严格控制压力，将压力控制在尽可能小的范围内，保证出土量与牵引量相匹配。同时尽量少做纠偏动作，即使做纠偏动作幅度也不宜过大。

# （7）导向钻进

采用精度为  $0.1\%$  的导向雷达控制钻进标高，导向标高控制在管中心线位置。钻杆轨迹的第一段是造斜段，控制钻杆的入射角度和钻头斜面的方向，缓慢给进而不旋转钻头，就能使钻头按设计的造斜段钻进。钻头到达造斜段完成处，接下来的是排水管流水段的钻进。旋转钻头，并提供给进力，钻头就能沿水平直线钻进，由于在钻头位置安装了最先进的探测仪器，在钻进过程中通过地面精密接收仪器，通过接收仪器数据调整钻头角度，使得钻进按照流水线标高路线前进。到达出口工作坑，完成钻孔工序。

# 1）钻进操作：

检查钻机的测量仪器的电源是否充足，不足的更换电池，特别是探头电池更不可马虎。按照测量仪器使用规范操纵仪器。探头装进探头盒内，打开接收机和同步显示器，转动探头盒检查仪器是否正常。把探头盒与钻头接好并连接在钻杆上，开机输送钻进液，检查钻头喷嘴是否通畅。启动机器进行造斜和水平导向钻进。

# 2）钻进操作需要注意的事项：

钻杆的上、下接头对正，边缓慢给进动力边慢转上扣，不得拧得过紧。

钻机就位要严格对中，保证钻机平稳、水平钻进。开钻前要认真检查钻杆，发现弯曲要及时调直或更换。

钻进速度不宜过快，应根据地层条件合理调定给进力。根据土层类别来确定钻进速度，在松砂层不宜超过  $3\mathrm{m / min}$ ，在硬土层中，钻进速度以钻机不发生跳动为宜。并经常检查钻头磨损程度，发现磨损严重，及时进行补焊修复。

造斜顶进时，每次顶进进尺控制在  $0.5\mathrm{m}$  左右为好，曲线要缓。

钻杆内不得进脏物，以免堵塞钻头喷嘴。

当机油堵塞报警指示灯亮时，说明油压不正常，要进行检查。

当机油堵塞报警指示灯亮起时，要更换机油滤芯。

按规范开动钻机后，不允许马上用全负荷钻进，必须先进行试运转，确定个部分运转正常后，方可开始钻进。

# 3）钻头位置检测

钻机随机配有一手提导航仪，用以确定钻头在地底的位置，检测钻头是否偏离设计轨迹。在造斜段，钻头每钻进  $10\mathrm{cm}$  就测一次钻头的位置，在平铺段每隔  $20\mathrm{cm}$  就检测一次。如果发现偏离轨迹，就通过调整钻头斜面的方向，进行纠偏。但纠偏不能太急，因为钻杆毕竟具有一定的刚度。应该在几根钻杆内完成纠偏，不能在一根钻杆内完成纠偏，同时也要注意不要纠偏过度。

# （8）回扩孔

导向孔钻进完毕后，装上各类回扩器，必须将钻孔扩大至适合管道铺设的直径。一般，在钻机对面的出口坑将回扩钻头连接于钻杆上，再回拉进行回扩，在其后不断地加接钻杆。根据导向孔与适合生产管铺设孔的直径大小和地层情况，

回扩可一次或多次进行。

钻机采用泥浆作为钻进液。利用泥浆混配系统将泥浆混合液通过钻杆注入回扩器中，通过回扩器的旋转均匀地喷在孔壁上。常用的泥浆（钻进液）为膨润土和水的混合物。

泥浆的主要作用：

1）泥浆能使钻杆和钻头得到良好的润滑，在道回拉时能润滑管道，减少阻力。  
2）泥浆在孔内流动，可冷却钻头。  
3）泥浆可悬浮岩粉，并将他们带出孔外。  
4）泥浆在钻头前面起射流作用，有助于切削土层。  
5）泥浆能辅助破碎地层、调整钻进方向。  
6）钻进岩石或其它硬地层时，可用钻进液驱动孔底泥浆马达，为泥浆马达提供动力。  
7）泥浆在孔的四周形成泥饼，也称之为孔壁，使形成的孔不塌方。

# （9）回拉敷设管道

在管头后端接上回扩头，管后接上分动器进行接管，将管子回接到工作井后，卸下回扩头、分动器、取出剩余钻杆，堵上封堵头，防止泥浆流出而影响路面沉降。

施工时，拉管机操作人员要根据设备数据均匀平稳的牵引管道，切不可生拉硬拽。

1）拉管时，需先检查机械固定措施是否牢固，不足部位及时调整；  
2）拉管时，机械操作员需高度注意力集中，随时观察机械扭矩等数据；  
3）拉管前，需联系好挖机，或钢索及卷扬机，以防拉力不够时，及时助拉；  
4）同时可以用挖机等推力机械从后将管材向前推等措施。

总之，回拖布管必须一气呵成，有一点疏忽将前功尽弃，使得整个布管失败，回拖布管成型后需及时将管头封堵，以防泥浆进入管内，堵塞管材。

# （10）注浆加固

牵引管成型后，成管与孔壁有一定的空隙，为保证以后不发生沉降、塌方等情况，应对牵引管孔内进行注浆，注浆材料为水泥浆。

注浆管塞入成孔内后，将成孔两端封堵并在另一端预留排气孔，用高压注浆泵将浆液注入孔中。待有反浆、外溢等现象后，结束注浆。

太湖流域地处长江三角洲的南翼，三面滨江临海，一面环山，北抵长江，东临东海，南滨钱塘江，西以天目山、茅山等山区为界，总面积  $36895\mathrm{km}^2$  。流域地形呈周边高、中间低的碟状，其西部为山区，中部为平原河网和以太湖为中心的洼地及湖泊，北、东、南周边受长江和杭州湾泥沙堆积影响形成地势高亢的碟边。流域水系以太湖为中心，分为上游水系和下游水系；上游主要为西部山丘区独立水系，有苕溪水系、南河水系及洮灞水系等；下游主要为平原河网水系，主要有以黄浦江为主干的东部黄浦江水系、北部沿江水系和南部沿杭州湾水系。京杭运河穿越流域腹地及下游诸水系，起着水量调节和承转作用。流域水利分区和水系分布见下图。

太湖流域是洪涝灾害频发区域，阳澄、淀泖和滨湖区同流域类似，一般是由夏季梅雨或夏秋季台风暴雨造成。解放以来，阳澄淀区发生洪涝灾害的年份有

1991年太湖大水之后，按照国家计委批复的《太湖流域综合治理总体规划方案》，流域综合治理十一项骨干工程相继开工建设，至2005年底全面建成，初步形成洪水北排长江、东出黄浦江、南排杭州湾充分利用太湖调蓄的流域防洪骨干工程体系框架，流域内的防洪除涝水环境和航运条件得到了较大改善，供水能力得到了一定的提高，流域初步具备防洪减灾、水资源优化配置、合理调度的基本条件。经过多年防洪体系的建设，流域防洪减灾能力得到一定程度提高。目前，流域具有了防御1954年实况洪水（相当于50年一遇）的能力，各地区防洪标准一般提高至10年~20年一遇。

# （2）武澄锡虞区概况

本项目电力管线线路均涉及武澄锡虞区。武澄锡虞地区位于太湖流域北部、江苏省南部，属太湖下游的低洼平原区，北依长江，南滨太湖，西界武澄锡西控制线与太湖湖西区相邻，东至望虞河东岸，行政区划涉及无锡、常州和苏州3市12个县市（区），包括无锡市区、江阴、张家港的全部和常州市区、常熟的一部分，区域总面积约  $4015.5\mathrm{km}^2$  。其中，无锡市境内面积约  $2261.6\mathrm{km}^2$  ，占  $56.3\%$  ；常州市境内面积约  $789.6\mathrm{km}^2$  ，占  $19.7\%$  ；苏州市境内面积约  $964.3\mathrm{km}^2$  占  $24.0\%$  。区内以白屈港东控制线为界，分为武澄锡低片及澄锡虞高片，其中武澄锡低片面

积  $2255.0 \mathrm{~km}^{2}$ , 澄锡虞高片（含沙洲独立排水片）面积  $1760.5 \mathrm{~km}^{2}$  。

武澄锡虞地区整体地形相对平坦，地势特点为四周较高、腹部低，形似“锅底”。境内地貌大部分属长江三角洲高亢平原、圩田平原和水网平原类型等，仅北端张家港市沿江地区属长江三角洲冲积平原区区内低山残丘主要分布在无锡市境内，市区西南部和江阴市北部的山丘总体上呈北东、北东东及近东西走向，最高峰为惠山的三茅峰，海拔  $328.98\mathrm{m}$  。区内水网平原区地面高程一般在 $3.5\sim 5.5\mathrm{m}$  （吴淞高程）之间，沿江高亢平原区地面高程在  $6\sim 7\mathrm{m}$  （吴淞高程）左右，低洼圩区地面高程一般在  $4.0\sim 5.0\mathrm{m}$  （吴淞高程），南端无锡市区及附近一带地面高程最低，仅  $2.8\sim 3.5\mathrm{m}$  （吴淞高程）。其中，武澄锡低片地形要比白屈港控制线以东的澄锡虞高片地形平均低  $1.5\sim 2.0\mathrm{m}$  （吴淞高程）。

根据地形特点与水系分布，境内水系总体以京杭大运河为界，分成运河北部水系和南部水系。北部水系以南北向通江河道为主，包括武澄锡低片的澡港、桃花港、利港、新沟河、新夏港、锡澄运河、白屈港和澄锡虞高片的走马塘、张家港、十一圩港以及以承担流域引排任务为主的望虞河等通江河道，同时西横河、黄昌河、应天河、青祝河、锡北运河、九里河、伯渎港等东西向河道与通江河道相连。南部水系主要以入湖河道为主，包括直湖港、武进港、梁溪河、曹王泾和大溪港等入湖河道，以及锡溧漕河、武南河、采菱港、永安河等内部骨干引排河道。京杭大运河自西向东经常州、无锡两市区贯穿区域内部，起着水量调节和承转的作用，并联接上述诸多河道，形成纵横交错、四通八达的河网。根据《江苏省骨干河道名录》（2018修订），武澄锡虞地区列入骨干河道名录河道共49条，其中流域性河道3条，总长  $223.6\mathrm{km}$  ；区域性骨干河道9条，总长  $320.8\mathrm{km}$  ；重要跨县河道9条，总长  $175.3\mathrm{km}$  ；重要县域河道29条，总长  $475.5\mathrm{km}$  。湖泊（湖荡）是本地区水系构成的重要组成部分。武澄锡虞地区除南滨太湖外区域内列入《江苏省湖泊保护名录》的主要湖泊（湖荡）尚有7处分别为五里湖、官塘、暨阳湖、宛山荡以及望虞河沿线的嘉菱荡、漕湖、鹅真荡，总面积约  $26\mathrm{km}^2$  ，其中嘉菱荡、鹅真荡及宛山荡为跨市湖泊。

1991年流域大水后，武澄锡引排工程相继启动实施，但受当时投资体制和经济条件等因素限制，建设标准普遍偏低，区域内其它河道治理以地方水利部门和企事业单位组织实施为主，建设标准和实施效果也参差不齐，且已整治河道经

过长期运行，由于岸坡坍塌和淤积等因素，外排能力减弱。在入湖控制严格的现状条件下，区域入江河道承担了更为主要的外排任务。锡澄运河黄昌河至长江段河道瓶颈仍未打通，白屈港、澡港河经长期运行，引排能力减弱，张家港、十一港等未规划安排进行治理，强降雨期间城市和圩区抽排的涝水无法通过区域骨干河道及时外排，极易发生洪涝灾害。2015~2017年区域河湖水位普遍超警戒、多处超历史，造成了较为严重的经济损失和社会影响，突出反映了区域外排能力存在的不足。

# 2.2.3 气象、水文

无锡市位于东经  $119^{\circ}31'$  至  $120^{\circ}36'$  、北纬  $31^{\circ}7'$  至  $32^{\circ}$  之间，地处长江三角洲江湖间走廊，江苏省的东南部，苏锡常地区之中部。

# （1）气候特征

据本工程沿线区域地处长江中下游地区，属北亚热带湿润季风气候区，四季分明热量充足，降水丰沛，雨热同季。夏季受来自海洋的夏季季风控制，盛行东南风，天气炎热多雨；冬季受大陆盛行的冬季季风控制，大多吹偏北风；春、秋是冬、夏季风交替时期，春季天气多变，秋季秋高气爽。通常在每年六月中旬到七月上旬前后将出现梅雨天气，是江南地区特有的气候特征，天气闷热、多雨、湿气较大。夏末秋初台风次数较多，往往带来狂风暴雨，具有较大的破坏性。常见的气象灾害有台风、暴雨、连阴雨、寒潮、冰雹和大风等。

# (2) 水文特征

无锡地区地处长江流域太湖水系区，区内地表水系极其发育，主要有京杭运河太湖支流及大小规模不等的河港沟塘组成。太湖水域面积为2250平方公里，总蓄水量在90亿  $\mathrm{m}^3$  左右；区内水网主要有太湖、盖湖、京杭运河、锡澄运河、咸塘河、黄杨巷浜、隔水河等。京杭运河、古运河、太湖湖群及河网，构成一个整体，其三者的水位动态变化密切相关，地处上游的长江、太湖湖群水位最高，其次为河网，呈同步升降，但幅度依次相应递减，大体反应了无锡地区地表水体

运动、变化规律。

河湖水位的变化与降水量年际、年内的变化基本一致，稍有滞后，由于地势低平水网水体径流迟缓，河床变化主要表现为淤积，水位变化于  $1.90 \sim 3.00 \mathrm{~m}$  之间，每逢汛期遭受洪水压力较大。从近几十年来资料（江苏省水文水资源勘测局无锡分局提供）反映，市区多年平均水位为  $1.25 \mathrm{~m}$ ，历史最高水位为2017年9月25日实测的  $3.36 \mathrm{~m}$ ，最低水位为  $0.104 \mathrm{~m}$  （1934年）（属1985年国家高程基准）。

无锡地区降水丰富，降雨集中在每年5~9月份的梅雨期与台汛期。在此之间易酿成洪涝灾害。例如1954年梅雨型洪水和1962、1991年的台汛型洪水，1954年5~7月份无锡市区降水量为  $798.5\mathrm{mm}$  （无锡地区降雨量  $1300\mathrm{mm}$  ），此次洪涝灾害特点为降水日多，降水总量大，太湖出现持续高水位（受浙西来水影响），同时长江中上游亦出现大洪水，使太湖流域排水严重受阻，京杭大运河无锡南门水位超过  $1.70\mathrm{m}$  警戒水位长达141天，7月28日当天最高水位达  $2.85\mathrm{m}$  。

# 2.2.4.2 区域地质条件

无锡市属长江三角洲的一部分，本区及边缘地段呈现一大碟形洼地，通称太湖区。该区域第四系沉积物具有以下特征：第四纪沉积物在基底隆起产生的次级凹陷和凸起相间的背景上堆积，沉降幅度相对较小，地势平坦，地表水系发育，第四系覆盖层厚度较大，厚度为  $40 - 197\mathrm{m}$ ，大多在  $80 - 130\mathrm{m}$  。地层结构以砂（砾）与黏性土互层为特征，韵律清晰，沉积相以河流、河湖相为主。本区晚更新世以来出现三次海侵，即全新世早期一次，晚更新世两次，其中以第二次湖海侵规模最大。分布于本区西南部和东北部低地丘陵区的第四系沉积物，受地形地貌的影响，沉积物质的成分和堆积方式与平原区不同，一般以洪坡积、残坡积、冲坡积、冲积形式混杂堆积，岩性复杂，地质环境条件复杂程度属中等地区。

本区的前第四纪地层，属扬子地层区下扬子地层分区宜兴-苏州地层小区。基岩露头主要分布在惠山、马山、军嶂山和查桥、张泾、八士、安镇等地，余者皆掩覆于第四系松散层之下，从老到新主要有泥盆系、石炭系、二叠系、三叠系中下统、侏罗系上统，白垩系和第三系。地表出露的地层主要为泥盆系石英砂岩、粉砂岩、泥岩等，碳酸盐岩类地层除查桥有零星出露外，其余均掩覆于第四系松散地层之下。

# 2.2.4.4水文地质条件

# 1、地表水

项目区属太湖水系，地表水体主要为锡澄运河。场地内河流水流平缓，基本无冲刷现象。根据本工程临近地表水化学分析结果，地表水对混凝土结构和对钢筋混凝土结构中的钢筋具微腐蚀性。

# 2、地下水

沿线地下水按埋藏条件分为潜水、微承压水、承压水。

# (1) 潜水

根据区域水文地质资料，潜水位一般在4~5月份随着降水量的逐渐增加，水位上升，至丰水期7~8月份出现高值，此后，降水量减少，水位缓慢下降，至次年1~3月枯水期出现低值，潜水位除受降水因素制约外，同时还受微地貌及地表水等因素的影响，一般在河间地段中间变幅小，沿河地段变幅大。总体来说，潜水位在一年当中均高于河水位，呈单向排泄于地表水。

本线路范围内潜水含水层主要为①1杂填土、①2素填土层，拟建项目区内均有分布，填土层由黏性土夹碎砖石等组成，由于其颗粒级配不均匀，固结时间短，往往存在架空现象而形成孔隙，成为地下水的赋存空间，其透水性不均匀。潜水埋深及水位受地形及地貌等因素的控制具有一定的变化，其主要补给源为大气降水、地表水入渗补给，主要以人工用水、蒸腾作用排泄。勘察期间测得潜水稳定水位埋深为地面下  $0.4\sim 1.8\mathrm{m}$  之间，潜水稳定水位标高  $1.03\sim 2.89\mathrm{m}$ ，平均水位标高  $1.80\mathrm{m}$  左右。根据区域水文地质资料，历史最高水位  $3.80\mathrm{m}$ ，近3~5年最高水位  $3.60\mathrm{m}$ ，水位年变幅约  $1.00\mathrm{m}$ ，地下水位标高随地形起伏有所变化（均为85国家高程）。

# (2) 微承压水

该含水层主要赋存于全新统中段河湖相为主的灰、灰黄色粉土及粉砂层中，富水性中等。其补给来源主要为上部潜水的垂直入渗及周围河（湖）水网的侧向补给。以民间水井用水及向周围河（湖）水网的侧向迳流或对深层地下水的越流为该含水层的主要排泄方式。

微承压水主要赋存于④1层粉土、④2层粉土中，勘察期间钻至该层，采用套管隔开其它含水层，并停钻8小时以上，测得微承压水稳定水位标高为 $1.2\sim 1.7\mathrm{m}$ 。本场地3~5年内最高微承压水水位标高  $1.6\mathrm{m}$  左右（均为85国家高程）。

# 3、腐蚀性评价

拟建场地地下水清澈、透明、无异味，场地内及附近无污染源。

根据《公路工程地质勘察规范》(JTGC20-2011)相关章节按II类环境评价标准判别：地表水对混凝土结构具微腐蚀性，在长期浸水下对混凝土中钢筋具微腐蚀性，在干湿交替下对混凝土中钢筋具微腐蚀性：地下水对混凝土结构具微腐蚀性，

在长期浸水下对混凝土中钢筋具微腐蚀性，在干湿交替下对混凝土中钢筋具微腐蚀性。

# 2.2.4.5 场地与地质地震效应

# 第3章 河道演变

# 3.1 河道历史演变概况

无锡历史悠久，早在6000多年前先民们就开始在这里劳动、生息和繁衍，并逐步开发水利，进行治水实践。大约4000多年前大禹在太湖地区治水，初步打开了太湖洪水入江入海通道，改变了太湖地区原始洪荒状态，为太湖地区的开发创造了条件。后经商周时期农田开发中的引排水治理、春秋至两汉时期以及唐宋时期的水利建设，到明清时期无锡地区的水系河网已基本形成和完善。

惠山区境域大部分地势低洼，境内高程在  $3.1\mathrm{m}$  以下的面积占较大比重。据水利志记载，古时低洼之处积水成湖，堪称水乡泽国：

西北部有浩渺的芙蓉湖，唐朝陆羽称其“南控长洲，东洞江阴，北掩晋陵，苍苍渺渺，近于轩户”，湖水迫近今无锡市区莲蓉桥附近，湖面“旧阔百里”，面积“万五千三百顷”。西部有阳湖，“东西八里，南北三十里”，东界一直到今安阳山麓。钱桥、洛社接壤处有限湖，“东西二十多里，南北四里多”。西漳、东北塘交界处有五步湖，“东西二十里，南北一十里，与芙蓉湖合抱芙蓉山，源浊而流清，溉田百余顷”。锡澄运河段则为水面辽阔的五泻水、白荡。

随着社会发展、人口增长，原有耕地不敷，便选择湖荡淤浅沮湿之地，筑堤围圩。可见，今西漳、堰桥、前洲、玉祁、石塘湾、洛社、钱桥、杨市等地，其低洼圩区之所在，是历代筑堤围垦建圩而来。圩外河道也随圩区的逐步形成而自然形成，从民国十二年制的无锡县水道图看，惠山区境内，圩外骨干河道之格局至今未有大的变化。

总之，惠山区的骨干河网水系已经基本形成，发生大规模自然演变的的机率很小，主要受人类活动的影响，如河道整治工程、清淤控制建筑物工程建设等。由于河道比降小，土质较差，极易发生淤浅的状况，整治疏浚是河网水系主要的演变特点。

R——水力半径（m）， $\mathrm{R} = \mathrm{A} / \chi$ ；

Q——洪峰流量（ $\mathrm{m}^{3} / \mathrm{s}$ ）；

I——河道比降；

# 4.3 冲刷淤积计算与河势影响分析

天然状况下，由于流域的来水、来沙及河床边界条件的不断变化，河床形态总是处在不断的冲淤变化过程当中。但在相当长的一个时段内，冲淤量可以相互补偿，河道处于一个相对的动力平衡状态。随着冲刷的发展，河床加深，过水面积加大，流速逐渐下降。

本次河床冲刷计算参考《公路工程水文勘测设计规范》（JTGC30-2015）中一般冲刷计算公式计算。

# 1、计算公式的选用及其适应性分析

计算河床冲刷，对水位、断面、河床质颗粒直径等，应符合下列要求：

（1）当桥梁位于天然冲淤或回水变化的河段时，应考虑水位的影响；  
(2) 计算断面应为天然情况下桥梁处计算冲刷不利的断面;  
（3）颗粒应能代表河床质的平均粒径，当粒径分布很不均匀时，应考虑河床质粗化的影响。

# 2、一般冲刷计算

河槽部分的一般冲刷:

$$
h _ {p} = \left[ \frac {A \frac {Q _ {2}}{\mu B _ {C}} \left(\frac {h _ {m c}}{h _ {c}}\right) ^ {5 / 3}}{0 . 3 3 \left(\frac {1}{I _ {L}}\right)} \right] ^ {5 / 8}
$$

式中：  $h_p$  ——河槽一般冲刷后最大水深，  $\mathrm{m}$

$A$  ——单宽流量集中系数，取  $1.0 \sim 1.2$ ；取值根据阚译先生《桥渡冲刷》中公式确定为 1.1；

$I_{L}$  ——冲刷坑范围内黏土液性指数，根据地勘报告物理力学性质指标统计表取值；

$Q_{2}$  ——河槽通过的设计流量， $\mathrm{m}^{3} / \mathrm{s}$ ，当河槽能扩宽至全桥时，取用设计流量  $Q_{p}$ ，当河槽不能扩宽时， $Q_{2} = \frac{Q_{c}}{Q_{c} + Q_{t}}. Q_{p}$ ；

$Q_{c}$  ——天然河槽流量  $(\mathrm{m}^{3} / \mathrm{s})$  ；

$Q_{t}$  ——天然状态河滩部分通过流量  $(\mathrm{m}^{3} / \mathrm{s})$  ；

$B_{C}$  ——河槽宽度,  $\mathrm{m}$ ;

$h_{mc}$  ——河槽最大水深，  $\mathrm{m}$ ;

# 4.4.2、现状护岸沉降分析

将穿越施工引起的地面沉降计算方法主要有经验法和解析法。目前工程界常采用 Peck 提出的地面沉降横向分布估算公式：

$$
\begin{array}{l} s _ {x} = s _ {\max } \cdot e ^ {- \frac {x}{2 t ^ {2}} ^ {2}} \\ s _ {\max } = v _ {1} / 2. 5 i \\ \end{array}
$$

式中  $\mathrm{S}_{\mathrm{x}}$  为地面沉降量（m）； $\mathbf{x}$  为离管道轴线的水平距离（m）； $\mathrm{S}_{\mathrm{max}}$  为管道轴线上方的最大地面沉降量（m）；i 为地面沉降槽宽度系数（m）； $\mathrm{V}_{1}$  为管道单位长度的土体损失量（ $\mathrm{m}^{3}/\mathrm{m}$ ），通常采用挖掘面面积的百分率来估算土体损失的大小，令  $\eta$  为土体损失百分率，则  $\mathrm{V}_{1} = \pi r^{2}\eta$ 。

$\boldsymbol{\eta}$  取值根据管道所处层位的土体性质确定，自稳性差、易流失的土取大值。

i的经验公式为：

$$
i = H / (2. 5 \cdot t g (4 5 ^ {\circ} - \varphi / 2))
$$

式中：H为管道埋置深度（m），本次取管道最小埋置深度9.509m；

# 5.3 建设项目对河道行洪的影响评价

# 5.3.1 项目对行洪的影响

本工程采用定向钻技术从锡澄运河河底以下穿越，管道埋于现状河床及规划河床泥面以下，施工过程中不断流，施工期和运行期不占用水域面积和过流断面，不会形成阻水、壅水，工程建设不影响锡澄运河的行洪、引排水。

施工期间所有弃土、钻渣、废弃泥浆，妥善暂存，最终运输至指定的处置地点，运输过程中不会遗洒，施工时禁止排入河道内，因此，不会对涉及河道防洪和引排产生影响。

# 5.3.2 洪水对项目的影响

根据第四章冲刷计算分析，管道穿越河道的冲刷深度小于水深，河流设计洪水位下，水流产生的一般冲刷不会对管道穿越工程造成影响。故洪水对项目无影响。

# 5.4建设项目对河势稳定的影响评价

本工程采用定向钻拖拉管下穿锡澄运河工程施工不会开挖河道断面，管道均位于现状及规划河底高程以下，不会在河道内增加阻水性建筑物，水流的流态不会因为管道工程的建设而发生变化，对锡澄运河河势的稳定不会产生影响。

通过对穿管处河槽的冲刷分析计算，得出在设计洪水情况下，河道冲刷深度小于水深，说明河道无一般冲刷影响，不会对穿越管道造成影响。

穿越河床管道所采用的管材接口采用直缝焊接，密封性较好。定向钻管道施工能有效控制出土量和地面沉降，采取向管外壁压注配置好的泥浆材料，以固化和密实管道周围局部扰动的土体，可有效使管壁与周围土壤的密实；且管道埋设于护岸以下位置较深，对护岸的稳定性无影响。

# 5.5建设项目对堤防安全及岸坡稳定和其他水利工程影响评价

本工程位置处锡澄运河（运东西漳大联圩-锡澄运河段）现状东岸为一级堤

防，为直立式挡墙护岸，堤防顶高程  $6.5\mathrm{m}$ ，防汛道路宽  $5 - 6\mathrm{m}$ ，为西漳大联圩的圩区堤防；本工程位置处锡澄运河西岸为混凝土自然土坡护岸三级堤防，惠澄大桥北侧为直立式挡墙护岸，作为石塘湾大联圩的圩区护岸。

惠澄大桥拆除重建后东西两侧新护岸均为A型直立式挡墙护岸，新建护岸进行打桩处理，护岸底桩基高程-8.194m，西侧与本次电力管道管顶最小距离为2.62m，东侧与本次电力管道管顶最小距离为3.0m，先进行电力管线施工，后进行护岸施工，护岸施工基本对本次电力管线不会有影响。

# 5.6建设项目对水利工程运行管理和防汛抢险的影响分析

项目建设区附近现状无大型水利工程和设施，目前项目附近无水利规划，不会对第三人合法水事权益造成影响。施工完成后应该清除施工产生的弃土弃渣，设备，恢复场地原貌，减少对防汛抢险造成影响，因此不会影响河道整治。

# 5.7 建设项目施工期影响评价

本工程采用定向钻施工，无需涉水施工，穿河段定向钻穿越牵引式施工的时候对河床产生甚微的扰动，不会对周边水环境造成影响，对河道两侧现有护岸基本无影响。

本工程穿越深度范围内存在②-1素填土、②-1淤泥质粉质黏土、④-1粉土、⑤-1粉质黏土，管道施工中可能引发的岩土工程问题主要是粉土、粉砂及粉细砂

的结构松散、成孔性差，孔壁容易坍塌。

其主要原因可能是：钻机能力不够；导向孔不符合要求；预扩孔偏移量过大；孔内泥眉堆积过多；塌孔；管道进孔不顺畅；设备故障。施工单位应采取相应的防范措施。

# 5.8建设项目对第三人合法权益的影响分析

本工程电力管下穿河道，建设完成后对河道行洪安全没有影响，由于本次惠澄大桥拆除重建，本次地埋管线均涉及迁改，因此本工程周边不存在平行穿越的地埋管线，仅存在垂直交叉的沿河管线。

管线北侧现状有锦通货运码头，锦通货运码头目处于正常运营状态，码头建有2个500吨级泊位，运营货种为石子和砂石，锦通货运码头与新建桥梁边缘相距  $23\mathrm{m}$ ，无锡市惠山区人民政府承诺在惠澄大桥工程开工前将完成码头拆迁工作，确保不涉及第三方权益方争议。

管线南侧约  $120 \mathrm{~m}$  处有一水闸，本次电力线穿越不会影响此处水闸的运行。

# 第6章 消除和减轻影响措施

# 6.1 建设项目消除和减轻影响的措施

针对工程建设对河段防洪所产生的影响，对施工期及项目建成后的正常运行期提出相应消除和减轻影响措施。

# 6.1.1 压力灌浆措施

拖拉管最终成孔直径为设计管径1.2~1.5倍，管道回拖完成后，管道四周尤其是上部将形成一定的间隙。为防止河床、堤防发生坍塌变形和形成渗水通道导向孔钻进、扩孔及回拖时，应及时向孔内注入钻进液；钻进液技术指标应满足《水平定向钻法管道穿越工程技术规程》；钻进液中膨润土掺入量暂按  $300\mathrm{kg / m}$  控制；加强拉管施工对地面的沉降、临近建（构）筑物、已有地下的管线影响的监测，以便于控制和防止地面沉降或隆起。

拉管结束后应用水泥浆置换触变泥浆和填充管周空隙，确保拉管管道与周边土层紧密结合，以提高管道四周土体的防渗性。

# 6.1.2 塌孔应急预案

对于地质因素造成的塌孔预防措施有：

（1）增大泥浆密度，提高泥浆液柱压力以平衡地层压力；  
（2）使用较大动塑比值的泥浆来保护孔壁。

对于物理因素造成的塌孔预防措施有：

（1）开泵避免过快，让钻杆尽量保持在恒定张力下工作，以避免液柱压力骤增以及钻具对孔壁的猛烈撞击；  
（2）根据地下障碍分布情况做好钻孔轨迹设计和扩孔程序，以避免因导向孔曲率过大引起的钻具回转挤压孔壁；  
(3) 合理控制泥浆密度和流变性能, 使泥浆具有一定的液柱压力, 保持泥浆的低返速, 呈平板流型的状态;  
(4) 提高钻进速度, 减少泥浆浸泡冲蚀时间, 同时保持泥浆均匀、性能稳定

定，严禁出现泥浆性能大幅度变化的情况；

配制泥浆应适当降低失水量，减少渗入孔壁的水分，适当增大泥浆滤液黏度， $\mathrm{pH}$  值控制在  $8.5 \sim 9.5$  之间；

当塌孔不严重时，适当增大泥浆密度，减少失水，小排量循环，使泥浆呈平板型层流或塞流，将塌块和岩屑带出。当塌块尺寸较大时，选择高切力、高黏度泥浆，保持泥浆在环形空间层流状态下，适当加大冲洗排量，以便将大塌块带出。

# 6.1.3 编制施工期应急预案

施工期应急预案应向相关主管部门予以报备，建设和施工单位应定期开展培训与演练，提高应急处置能力。

预案中明确了事故响应与应急处置程序和措施，成立了应急指挥部和各救援小组，配备了各项应急抢险物资。一旦发生险情，立即上报，根据情况启动应急预案，各救援组按照责任分工，开展防汛抢险工作。

应急措施启动后，项目部应急救援领导小组副组长根据组长的部署，有效组织组织调配、征用本单位或协作单位抢险救援队伍，调配应急救援物资、药品、医疗器械、救援船舶、挖掘机、起重机械、编织袋、砂子、救生衣和救生圈、照明设备、抢险车辆等物资，及时对事故现场进行应急救援。

# 6.1.4 设置管道标志牌

本工程管线标志需设置在河流两岸上下游适当位置，并在管线正上方设置标志牌，标志牌中除标明联系方式等基本信息外，还应注明管线保护范围，路由走向，管道穿河断面和管顶高程等内容

与本工程同步设计、同步施工、同步维护。

# 6.1.5 定期监测

建议定向钻施工过程中，委托第三方对堤岸进行位移和沉降监测，防止管涌等情况发生。管道运行期间定期进行检测，发现问题需在水行政主管部门的指导下采取处理措施，确保水利工程的安全、稳定。

# 第7章 结论与建议

# 7.1 防洪综合评价主要结论

# 1、项目建设方案基本符合相关规划要求

本工程采用定向钻拖拉管穿越锡澄运河，本工程建设不影响相关水利规划的实施，与相关规划相符。

# 2、建设项目符合相关技术要求

通过采取消除与减轻影响措施，本项目下穿锡澄运河工程建设满足技术规定要求。

# 3、建设项目对行洪无影响

拟建管道工程从河道底部穿越，不改变现状河道行洪断面，工程运行期对河道行洪、引排水均没有影响。

# 4、建设项目对河势稳定没有影响

本工程涉及河道经过长期自然演变和一系列人类活动后，河势较为稳定。本工程管道埋设深度较深，不存在上浮风险，建设项目对河势稳定无影响。

# 5、建设项目对堤防安全及岸坡稳定和其他水利工程影响较小

本工程的建设对现有防洪工程不会造成影响，本工程建成后，建设单位应将本工程管道的出入土点坐标、管道轨迹，管顶高程、地面高程等技术参数同时提交相关设计单位与水利局。

# 6、建设项目对水利工程运行管理和防汛抢险没有影响

项目建设区附近现状无大型水利工程和设施，目前项目附近无水利规划，不会对第三人合法水事权益造成影响。施工完成后应该清除施工产生的弃土弃渣，设备，恢复场地原貌，减少对防汛抢险造成影响，因此不会影响河道整治。

建设项目对防汛抢险无影响。

# 7、建设项目施工期影响较小

通过采取泥浆处置措施后，本工程施工期对河道影响较小。

# 8、对第三人合法水事权益影响较小

本工程的建设对第三人合法水事权益无影响。

# 7.2 消除和减轻影响措施结论

拉管结束后应用水泥浆置换触变泥浆和填充管周空隙，确保拉管管道与周边土层紧密结合，以提高管道四周土体的防渗性。

通过采取压力灌浆、塌孔应急预案等措施，能减轻定向钻对河道及堤防的影响。

# 7.3 建议

# 1、规范施工，及时上报

施工期须严格按照施工方案及有关施工规范进行，以避免对水利工程产生不利影响。施工期若河道岸坡出现变形、沉降、坍塌等情况，应及时报告当地水行政主管部门，在其指导下采取相应措施。

# 2、弃土弃渣，妥善处置

施工期间所有弃土、钻渣、废弃泥浆需规范弃置在落实好的处置地点，禁止就近排入河、湖内。

# 3、设置管道标志

河道管理范围内的下穿管线应当在合适的位置设置永久性的识别标志，并标明相关信息。

# 4、加强巡查，接受督察

加强运行期管理，制定巡查制度，特别是汛期要加强对管道穿越段及其上下游河岸的巡查。

# 5、上报竣工资料

工程竣工后，建设单位须向水利局提交有资质单位测绘的工程竣工图及相关竣工资料归档，明确竣工后实际管道走向及埋深。

# 省发展改革委关于312国道无锡凤翔路至广石路段工程初步设计的批复

无锡市发展改革委：

《关于上报312国道无锡凤翔路至广石路段工程初步设计的请示》（锡发改基础〔2023〕7号）及相关附件收悉。结合省交通运输厅出具的《关于312国道无锡凤翔路至广石路段初步设计的审查意见》（苏交计〔2023〕44号），经研究，现对该项目初步设计批复如下：

# 一、路线走向

本项目路线起自现状312国道与惠山区凤翔路交叉处西侧，沿老路线位向西，跨229省道、锡澄运河，止于广石路西侧，全

长4.106公里。下阶段做好起终点与相邻路段的衔接，进一步优化平、纵线形设计，完善标志标线等交通安全设施设计。

# 二、主要技术标准

原则同意本项目采用设置辅路的一级公路标准建设，主路为双向六车道，设计速度80公里/小时；辅路为双向六车道，设计速度50公里/小时。新建桥涵设计汽车荷载为公路-I级，其他技术指标按交通运输部颁《公路工程技术标准》（JTG B01-2014）和《城镇化地区公路工程技术标准》（JTG 2112-2021）的有关规定执行。

# 三、路基、路面及排水

同意初步设计推荐的路基标准横断面、一般路基设计方案和特殊地基处理方案。下阶段要结合详勘成果，优化软基处理方案，加强老路改造利用方案研究。

同意初步设计推荐的路面结构设计方案。下阶段要结合项目区域工程条件与沿线筑路材料的实际情况，总结已建公路经验，充分吸收应用相关科研成果，并根据交通量及交通组成，进一步优化路面结构设计。

同意初步设计推荐的路基、路面排水及路基防护设计方案。下阶段加强路基路面综合排水设计。

# 四、桥梁

原则同意初步设计推荐的全线桥梁设计方案。全线设置主线桥梁1座，桥长3605米；辅道桥梁4座，总长946.8米（大桥2座、桥长862.8米，中桥2座、桥长84.0米）。下阶段要结合区域路网

情况及详勘资料，进一步优化桥梁设计，合理确定桥梁规模，重点加强桥梁标准化设计和结构抗震设计。

# 五、交叉

原则同意初步设计推荐的全线交叉设计方案。下阶段要根据预测的转向交通量，结合地形地物等控制因素，深化交叉设计方案，做好平交口渠化设计及交通安全设施设计，保障行车安全。

# 六、交通工程及沿线设施

原则同意初步设计推荐的全线交通工程及沿线设施设计方案。下阶段要进一步做好超限检测、路网监测等设施设计，并与主体工程同步实施。

# 七、工程用地

本项目用地按照《江苏省自然资源厅关于312国道无锡凤翔路至广石路段项目用地预审意见》（苏自然资预〔2022〕51号）明确的9.5787公顷控制。下阶段要按照国家有关工程用地标准，进一步优化完善相关工程设计方案，从严控制用地数量。

# 八、建设工期

本项目建设工期为3年。

# 九、工程概算

核定初步设计概算265511.82万元，其中建筑安装工程费182760.82万元、土地使用及拆迁补偿费59073.72万元、工程建设其他费11061.55万元、预备费12615.73万元。项目资金来源按我委苏发改基础发〔2023〕285号文执行。

# 十、其他

下阶段要落实绿色公路、品质工程建设的相关要求，做好项目系统设计、安全设计、生态环保设计，提升工程安全保障水平和绿色环保水平。

请依职责加强对工程建设的监督，督促项目法人认真贯彻项目法人责任制、招标投标制、工程监理制、合同管理制等规定，切实防范政府债务风险，狠抓“质量、工期、投资”三大控制，做好施工图设计，及时办理开工前的各项手续。要加强安全生产管理，严格执行“三同时”制度，按照相关规章制度压实项目建设单位与相关责任主体安全生产及监管责任，落实安全生产措施，严防安全生产事故，确保工程建设顺利进行和社会稳定，力争早日建成通车。
